# `operations` command

    >>> run("vml operations --help")
    Usage: vml operations [OPTIONS] [FILTER]...
    <BLANKLINE>
      Show available operations.
    ...
    <exit 0>

The alias `ops` may also be used.

    >>> run("vml ops --help")
    Usage: vml ops [OPTIONS] [FILTER]...
    <BLANKLINE>
      Show available operations.
    ...
    <exit 0>
